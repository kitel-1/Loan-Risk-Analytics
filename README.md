# 个人信用贷款风控数据分析
## 📋 项目简介
本项目是**银行个人信贷信用风险全流程分析**，基于真实信贷用户数据，通过数据清洗、风控指标计算、好坏贷款划分、多维度可视化分析，完成个人贷款信用风险评估、客群资质分层、资产质量监控，完整复刻银行贷前审核、贷中额度管控、贷后风险预警的核心风控工作。

## 🎯 项目背景
个人信用贷款是银行核心业务，但用户逾期违约会造成直接资金损失。
本项目通过数据分析**精准识别高风险违约客群、量化还款能力、评估信贷资产健康度**，为银行信贷审批、风险管控、资金投放提供数据支撑。

## 📊 数据集说明
数据集包含个人信贷核心特征，无时间/地理字段，核心字段如下：
- 基础信息：`Loan ID`、`Annual Income`、`Monthly Debt`、`Years in current job`
- 资产信息：`Home Ownership`、`Current Loan Amount`、`Current Credit Balance`
- 征信信息：`Credit Score`、`Months since last delinquent`、`Number of Credit Problems`
- 风险标签：`Bankruptcies`、`Tax Liens`、`Purpose`

## 🔄 项目核心流程
### 1. 数据清洗与预处理
- 剔除异常值（超高信用分、天价贷款等脏数据）
- 处理缺失值（`Months since last delinquent` 空值 = 无逾期记录）
- 数据标准化，保证分析准确性

### 2. 风控核心指标计算
- **DTI 负债收入比**：`每月负债 / 月收入 × 100%`（银行风控红线指标）
- 累计放款金额、回款金额、优质/坏账贷款占比

### 3. 好坏贷款自动划分
✅ **优质贷款**：无逾期、无信用问题、无破产、无欠税记录
❌ **坏账贷款**：存在逾期、信用污点、破产、税务违约任意一项

### 4. 多维度可视化风控分析
基于清洗后数据完成**6大专业可视化图表**，全维度拆解信用风险规律：
1. 信用分数 vs 贷款金额 散点图
2. 住房类型与贷款金额分布 箱线图
3. 年收入-月负债-DTI 风险气泡图
4. 工作年限与放款规模 柱状图
5. 贷款用途放款金额分析图
6. 住房类型资金占比 矩形树图

## 📈 核心分析维度
1. **客群稳定性分析**：工作年限越长，职业稳定性越高，放款规模越大
2. **资产资质分析**：自有房产/按揭房客群为优质低风险群体，租房客群风险偏高
3. **还款能力分析**：DTI 指标直接反映还款压力，是逾期违约核心预判依据
4. **征信资质分析**：信用分数与贷款额度正相关，低分客群违约风险显著提升
5. **资产质量分析**：统计优质贷款/坏账贷款占比，评估银行整体信贷健康度

## 🔍 关键风控结论
1. **工作稳定性是风控核心**：10年以上工龄客群为银行最优放贷对象，违约风险最低
2. **房产资质决定风险等级**：自有房 > 按揭房 > 租房，房产越优质，贷款额度越高、风险越低
3. **DTI 是逾期预警黄金指标**：高负债低收入客群为坏账重灾区
4. **信贷资产结构健康**：优质贷款占比高，银行资金主要投向低风险优质客群
5. **异常数据管控**：剔除极端异常值后，分析结果完全符合银行真实风控逻辑
